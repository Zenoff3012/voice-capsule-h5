# 图标文件

本目录存放PWA图标，需要准备以下尺寸：

## 必需尺寸

| 尺寸 | 用途 |
|------|------|
| 72x72 | 安卓启动图标 |
| 96x96 | 安卓启动图标 |
| 128x128 | Chrome Web Store |
| 144x144 | 微软磁贴 |
| 152x152 | iPad启动图标 |
| 192x192 | 安卓启动图标、PWA |
| 384x384 | 安卓启动图标 |
| 512x512 | 安卓启动图标、PWA |

## 图标设计建议

1. **主色调**：橙色（#f97316）
2. **设计元素**：可以用心形、声波、礼物盒等元素
3. **文字**：可以加上"亲"字
4. **格式**：PNG，透明背景

## 快速生成

可以使用以下工具生成：
- [PWA Asset Generator](https://pwa-asset-generator.nicepkg.cn/)
- [Figma](https://figma.com) 设计后导出
- [Canva](https://canva.com) 在线设计

## 文件名规范

```
icons/
├── icon-72x72.png
├── icon-96x96.png
├── icon-128x128.png
├── icon-144x144.png
├── icon-152x152.png
├── icon-192x192.png
├── icon-384x384.png
└── icon-512x512.png
```

## 临时方案

如果没有设计好的图标，可以先用以下方式生成简单图标：

1. 使用在线工具：[favicon.io](https://favicon.io/)
2. 文字图标：输入"亲"字，选择橙色背景
3. 下载后按尺寸重命名

> 注意：正式上线前建议请设计师制作专业图标
